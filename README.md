# ge2rb-guide
This is an interactive guide for the game God Eater 2 Rage Burst.

## Current Features

- Aragami List
  - attribute
  - weaknesses
  - resistances
  - weakspots
  - breakable parts
  - drop prefixes
- Skills List
  - Skill Effects
  - Support Skills
  - Compound Skills
